# Security Policy

## Supported Versions

`wh-cli` has not reached a stable release yet. Security fixes are applied to `main` until the first versioned release exists.

## Reporting a Vulnerability

Report security issues privately by email:

security@example.com

Include:

- affected version or commit
- operating system
- reproduction steps
- impact assessment
- whether secrets, WhatsApp sessions, or message contents may be exposed

## Disclosure

We aim to acknowledge reports within 72 hours and publish a fix or mitigation timeline after triage. Please do not open public issues for vulnerabilities involving auth, encrypted storage, keyring handling, logs, or session data.

## PGP

A project PGP key will be published before the first public release. Until then, use email and avoid sending live WhatsApp sessions or production secrets.
